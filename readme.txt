This Eclipse Android project is intended to be used as a skeleton or template 
to learn the usage of Android JUnit test framework or to check the availability 
of it with your development environment. You can find the original code on Github
https://github.com/TakashiSasaki/HelloAndroid.git .

It is the target project used with HelloAndroidTest project on Github.
https://github.com/TakashiSasaki/HelloAndroidTest.git .
